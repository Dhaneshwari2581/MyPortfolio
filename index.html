Vehicle BlackBox for accident detection and sms alert

Technologies: Arduino / Microcontroller , GPS Module, GSM Module, Embedded C Programming

The Vehicle Black Box for Accident Detection and SMS Alert System is an embedded system designed to automatically detect vehicle accidents and immediately send an emergency SMS alert with location details to predefined contacts. This system helps in reducing emergency response time and improving road safety.

Key Features

  • Real-time accident detection using vibration/impact sensor
  • Automatic SMS alert to emergency contacts
  • GPS location tracking
  • Data logging for post-accident analysis
  • Compact and cost-effective design
